# Chuffed-Guides

Personal documentation guides. Privacy, Linux, finance, and other topics added over time.

Live site: https://chuffed-code.github.io/Chuffed-Guides/

## Stack

Built with [Hugo](https://gohugo.io/). Deployed to GitHub Pages via GitHub Actions.

## Local development

Requires Hugo (extended).

```bash
brew install hugo
hugo server
```

Site renders at http://localhost:1313.
